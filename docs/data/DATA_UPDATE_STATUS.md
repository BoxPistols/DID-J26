# データ更新状況: 2020年 DIDデータ

## 概要
2010年（平成22年）のDIDデータから、2020年（令和2年）データへの移行を進めています。
e-Statからのデータ一括ダウンロードが困難なため、手動ダウンロードとスクリプトによる変換を組み合わせて対応します。

## 現状のステータス
- **北海道 (01):** 2020年データへ更新済み (`public/GeoJSON/2020/r02_did_01_hokkaido.geojson`)
- **その他 (02-47):** 未更新 (2010年データを使用中)

## 更新手順
残りの都道府県データを更新する場合は、以下の手順を実行してください。

1. **ダウンロード**
   - [e-Stat 地図データダウンロード](https://www.e-stat.go.jp/gis/statmap-search?page=1&type=2&aggregateUnitForBoundaryOn=A) にアクセス
   - `小地域` > `国勢調査` > `2020年` > `人口集中地区` > `世界測地系緯度経度・Shapefile` を選択
   - 必要な都道府県のZIPファイルをダウンロード

2. **ファイル配置**
   - ダウンロードしたZIPファイルをそのまま `rawdata/2020/` ディレクトリに配置してください。
   - ファイル名変更は不要です。

3. **変換実行**
   以下のコマンドを実行すると、`rawdata/2020/` 内のZIPファイルを自動検出し、GeoJSONに変換して `public/GeoJSON/2020/` に配置します。
   ```bash
   npm run convert:did
   ```

4. **確認**
   - `public/GeoJSON/2020/` に新しいGeoJSONファイルが生成されていることを確認してください。
