# DIDデータ最新化 要件定義

## 概要

現在のデータ（平成22年/2010年）を令和2年（2020年）国勢調査データに更新する。

## ステータスと手順
**更新作業中** です。詳細は [DATA_UPDATE_STATUS.md](./DATA_UPDATE_STATUS.md) を参照してください。

- **北海道 (01):** 更新完了
- **その他:** 未更新（順次対応）

## 目標

- データ年次: 令和2年（2020年）
- ファイル形式: GeoJSON（現行維持）
- 座標系: EPSG:4326（現行維持）

## 技術スタック変更点
当初 `ogr2ogr` (GDAL) の使用を予定していましたが、環境構築の容易さから Node.js ベースの `mapshaper` を使用する方針に変更しました。
